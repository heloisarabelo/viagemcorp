☁ Banco de Dados na Nuvem (Supabase)
Acesse supabase.com → Projeto → Project Settings → API para obter as credenciais.
Precisa criar a tabela no Supabase? Va em SQL Editor e execute:
create table if not exists vc_state (
  id text primary key,
  data text,
  updated_at timestamptz default now()
);
alter table vc_state enable row level security;
create policy "Allow all" on vc_state for all using (true) with check (true);
